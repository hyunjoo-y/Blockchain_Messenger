# 📩 Blockchain Messenger_Kotlin_v0.1

---

## 📌 프로젝트 개요
Blockchain Messenger는 **블록체인 기반 사용자 정보 관리 및 libp2p를 활용한 P2P 채팅 기능**을 제공하는 탈중앙화 메신저 애플리케이션입니다. Android 환경에서 실행되며, 사용자(회원) 관련 메타 데이터는 블록체인에 저장되어 보안성을 높이고, 개인 데이터는 사용자 디바이스에 저장됩니다. 중앙 서버 없이도 안전한 P2P 메시지 전송이 가능합니다. 

---

## 🌟 주요 기능
✅ **블록체인 기반 사용자 정보 관리** (Ethereum 스마트 컨트랙트 활용)  
✅ **P2P 메시징 기능** (`libp2p` 사용)  
✅ **안드로이드(Android, Kotlin) 기반 개발**  
✅ **친구 추가/삭제 기능**  
✅ **서버리스 메시징 시스템**

## ⚙️ 기술 스택
- **블록체인**: Ethereum
- **스마트 컨트랙트**: Solidity  
- **프로그래밍 언어**: Kotlin
- **데이터 저장소**: IPFS(프로필 정보), 개인 디바이스(중요한 개인 데이터)
